## 📖 项目简介
这是一个功能完整的企业ERP系统，专为中小企业设计，提供全面的业务管理功能，包括进销存、财务管理、生产管理等模块，帮助企业实现信息化管理，提高运营效率。

## 🚀 系统功能

### 导航栏结构
系统采用清晰的导航栏分层结构，方便用户快速找到所需功能：

1. **首页**
   - 系统概览：展示企业运营关键指标和数据统计

2. **零售管理**
   - 零售出库、零售退货

3. **采购管理**
   - 请购单、采购订单、采购入库、采购退货

4. **销售管理**
   - 销售订单、销售出库、销售退货

5. **仓库管理**
   - 其他入库、其他出库、调拨出库、组装单、拆卸单

6. **财务管理**
   - 收入单、支出单、收款单、付款单、转账单、收预付款

7. **报表查询**
   - 商品库存、账户统计、零售统计、采购统计、销售统计、入库明细、出库明细、调拨明细、入库汇总、出库汇总、进销存统计、客户对账、供应商对账、库存预警

8. **商品管理**
   - 商品类别、商品信息、多单位、多属性

9. **基础资料**
   - 供应商信息、客户信息、会员信息、仓库信息、收支项目、结算账户、经手人管理

10. **系统管理**
    - 角色管理、用户管理、机构管理、日志管理、商品属性、系统配置、功能管理、租户管理、插件管理、平台配置

### 核心功能详解

#### � 零售管理
- **零售出库**：直接面向终端客户的销售，支持快速开单、扫码录入等便捷操作
- **零售退货**：处理零售客户的退货请求，确保客户满意度

#### 📦 采购管理
- **请购单**：发起采购需求，规范采购流程
- **采购订单**：与供应商签订采购协议，明确采购数量、价格和交付时间
- **采购入库**：将采购的物料录入仓库，更新库存数量
- **采购退货**：处理不合格物料的退回流程，维护企业权益

#### 💰 销售管理
- **销售订单**：与客户签订销售协议，明确销售数量、价格和交付时间
- **销售出库**：将物料发给客户，减少库存数量
- **销售退货**：处理客户退回的物料，增加库存数量

####  仓库管理
- **其他入库**：处理非采购渠道的商品入库业务，如赠品入库、盘盈入库等
- **其他出库**：处理非销售渠道的商品出库业务，如样品出库、盘亏出库等
- **调拨出库**：在不同仓库之间转移商品，优化库存配置
- **组装单**：将多个零部件组装成一个成品，满足生产需求
- **拆卸单**：将一个成品拆分为多个零部件，灵活应对市场变化

#### 💵 财务管理
- **收入单**：记录企业的非销售类收入，如利息收入、租金收入等
- **支出单**：记录企业的非采购类支出，如办公费用、差旅费等
- **收款单**：记录企业收到的款项，如收到客户的货款、预付款等
- **付款单**：记录企业支付的款项，如支付供应商的货款、预付款等
- **转账单**：记录企业内部账户之间的转账业务
- **收预付款**：记录企业收到的客户预付款

#### 📈 报表查询
- **商品库存**：查询当前各商品的库存状况，包括库存数量、库存金额等信息
- **账户统计**：查询企业各账户的资金变动情况
- **零售统计**：查询企业的零售业务数据
- **采购统计**：查询企业的采购业务数据
- **销售统计**：查询企业的销售业务数据
- **入库明细/汇总**：查询商品入库的详细记录和汇总数据
- **出库明细/汇总**：查询商品出库的详细记录和汇总数据
- **进销存统计**：查询商品的进、销、存情况
- **客户/供应商对账**：与客户或供应商核对往来账款
- **库存预警**：查询库存低于安全库存的商品

#### 🛍️ 商品管理
- **商品类别**：对商品进行分类管理，支持多级分类
- **商品信息**：维护企业的商品档案，包括商品名称、规格、单位、价格等信息
- **多单位**：设置商品的多种计量单位，如件、箱、吨等
- **多属性**：设置商品的多种属性，如颜色、尺寸等

#### 📋 基础资料
- **供应商信息**：维护企业的供应商档案
- **客户信息**：维护企业的客户档案
- **会员信息**：维护企业的会员档案
- **仓库信息**：维护企业的仓库档案
- **收支项目**：维护企业的收支类别
- **结算账户**：维护企业的银行账户和现金账户信息
- **经手人管理**：维护企业的业务经手人信息

#### ⚙️ 系统管理
- **角色管理**：定义系统中的角色类型和权限配置
- **用户管理**：管理系统用户账号，包括增删改查、权限分配等
- **机构管理**：维护企业的组织机构信息，包括部门、岗位等
- **日志管理**：记录用户在系统中的操作行为
- **商品属性**：管理商品的特殊属性
- **系统配置**：设置系统的各项参数
- **功能管理**：管理系统功能模块
- **租户管理**：管理系统租户
- **插件管理**：管理系统插件
- **平台配置**：设置平台级别的配置
