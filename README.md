## 📖 项目简介
这是一个功能完整的企业ERP系统，专为中小企业设计，提供全面的业务管理功能，包括进销存、财务管理、生产管理等模块，帮助企业实现信息化管理，提高运营效率。

## 🚀 系统功能

### 导航栏结构
系统采用清晰的导航栏分层结构，方便用户快速找到所需功能：

1. **首页**
   - 系统概览：展示企业运营关键指标和数据统计

2. **物料管理**
   - 物料分类、物料属性、物料信息、物料特性管理

3. **单据管理**
   - 采购管理：采购申请、采购订单、采购入库、采购退货
   - 销售管理：销售订单、销售出库、销售退货
   - 零售管理：零售出库、零售退货
   - 仓库管理：其他入库、其他出库、调拨出库、物料组装、物料拆卸

4. **财务管理**
   - 收款、付款、预收款、转账、收入、支出管理

5. **报表查询**
   - 采购报表：采购入库、采购退货报表
   - 销售报表：销售出库、销售退货报表
   - 零售报表：零售出库、零售退货报表
   - 库存报表：物料库存、出入库明细、物料出入库统计、库存预警报表
   - 财务报表：账户报表、客户账户、供应商账户

6. **系统管理**
   - 基础数据：客户、供应商、仓库、单位、收支项目管理
   - 用户权限：用户、角色、权限、组织架构管理
   - 系统配置：系统参数、平台配置、插件、租户管理
   - 日志监控：日志、消息、会员管理

### 核心功能详解

#### 🔧 物料管理
- **物料分类**：创建多层级的物料分类结构，便于管理和查询
- **物料属性**：定义物料的基本属性（如颜色、尺寸等），实现精细化管理
- **物料信息**：维护物料的详细信息，包括编码、名称、规格、成本价、售价等
- **物料特性**：管理物料的特殊属性和参数，满足不同行业需求

#### 📦 采购管理
- **采购申请**：发起采购需求，规范采购流程
- **采购订单**：与供应商签订采购协议，明确采购数量、价格和交付时间
- **采购入库**：将采购的物料录入仓库，更新库存数量
- **采购退货**：处理不合格物料的退回流程，维护企业权益

#### 💰 销售管理
- **销售订单**：与客户签订销售协议，明确销售数量、价格和交付时间
- **销售出库**：将物料发给客户，减少库存数量
- **销售退货**：处理客户退回的物料，增加库存数量

#### 🛒 零售管理
- **零售出库**：直接面向终端客户的销售，支持快速开单
- **零售退货**：处理零售客户的退货请求，确保客户满意度

#### 📊 仓库管理
- **其他入库**：记录非采购渠道的物料入库（如盘盈、捐赠等）
- **其他出库**：记录非销售渠道的物料出库（如盘亏、内部领用等）
- **调拨出库**：在不同仓库之间转移物料，优化库存配置
- **物料组装**：将多个物料组合成一个成品，满足生产需求
- **物料拆卸**：将一个成品拆分为多个物料，灵活应对市场变化

#### 💵 财务管理
- **收款管理**：记录收到的客户款项，及时更新客户账户
- **付款管理**：记录支付给供应商的款项，维护良好的合作关系
- **预收款管理**：管理客户的预付款项，提高资金使用效率
- **转账管理**：处理账户之间的资金转移，方便资金调配
- **收入支出**：记录企业的各项收入和支出，掌握财务状况

#### 📈 报表查询
- **采购报表**：分析采购情况，优化采购策略
- **销售报表**：了解销售业绩，制定销售计划
- **零售报表**：掌握零售业务，调整零售策略
- **库存报表**：监控库存状况，避免库存积压或短缺
- **财务报表**：分析财务数据，支持决策制定

#### ⚙️ 系统管理
- **基础数据**：维护企业运营所需的基本信息
- **用户权限**：控制系统访问权限，确保数据安全
- **系统配置**：根据企业需求定制系统功能
- **日志监控**：记录系统操作，便于问题追踪和性能优化
